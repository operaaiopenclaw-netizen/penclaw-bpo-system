# 00 - Inbox

**Propósito**: captura sem atrito. Tudo o que não tem casa clara vai aqui primeiro.

## Regras

1. Criar nota rápida aqui **não precisa de template**.
2. Nome do arquivo pode ser horrível (ex: `ideia 3`, `???`). Triar depois.
3. Máximo 7 dias no Inbox — senão: promover, arquivar, ou deletar.
4. Na triagem, cada item vira uma de 4 coisas:
   - Promovido para nota estruturada em pasta própria
   - Virou tarefa no ORKESTRA (não mora mais aqui)
   - Arquivado em `99 - Archive/` com prefixo de data
   - Deletado (era ruído)

## O que vai aqui

- Insight de reunião que ainda não foi pensado
- Print do Telegram com questão aberta
- Link de artigo / vídeo / benchmark
- Ideia maluca para produto
- Observação do operador "fornecedor X demorou hoje"
- Pergunta que alguém fez e não teve resposta

## O que **não** vai aqui

- Alertas operacionais (esses já estão no ORKESTRA)
- Tarefas de projeto em andamento (use issue tracker)
- Rascunho de decisão já debatida (vai direto em `09 - Decisions`)

## Triagem — checklist

- [ ] Abrir Inbox toda segunda-feira (parte do weekly review)
- [ ] Para cada item: **é aprendizado, decisão, ou SOP-candidato?**
- [ ] Mover/renomear conforme a convenção
- [ ] Adicionar tags mínimas antes de mover
- [ ] Inbox vazio ao fim da segunda = meta

Ver `[[INTEGRATION]]` para como converter captura em nota final.
